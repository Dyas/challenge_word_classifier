Исходниуи состоят из 4 файлов:

1. prepare-dict.js - файл для формирования из исходного словаря - нового, усеченного словаря.
Для усечения используются алгоритм стемминга Портера и эвристики, полученные при помощи статистического анализа

2. gen.js - файл для генерации хэшей и упаковки их в data файл, который в дальнейшем поставляется с решением.
Для хэша используется стандартная функция для строк - ly. Хэши усекаются по модулю

3. bloom.js - файл содержит методы для подготовки и упаковки слова в data файл

4. solution.js - файл содержит алгоритм стемминга и подготовки слов при помощи эвристик.
Так же здесь содержится определение хэш-функции и проверки наличия хэша в файле data

Алгоритм предполагает использование фильтра блума с предподготовленными словами.
Кроме того само слово дробится на куски и записываются хэши для каждого из кусков.
Размер куска подобран эмпирически. Модуль усечения хэша подобран эмпирически.
