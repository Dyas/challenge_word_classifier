мой генератор считывает слова и записывает их по частям крытным 3 буквам отбрасывая лишнее

пример constrictors

sym 0 ,con,  // если в эту ячейку con уже записан то повторно записи не будет
sym 1 ,str,
sym 2 ,ict,
sym 4 ,ors,

и так же записывает переходы co[ns]t[ri]c[to]rs

con 1 ,ns,
con 2 ,ri,
con 3 ,to,

для того что бы уменьшить вероятность неверных связей ячеек

слова до 3 букв апрувятся все состоящие из алфавита 26 букв и '
исключая ' в первом символе слова
